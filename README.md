# Фильтр вкладов

Программа на C++ обрабатывает массив данных о вкладах и выводит только те значения, которые больше или равны 100 долларам.

## Возможности
- Фильтрация вкладов от $100 и выше.
- Вывод значений в удобном формате (`значение$`).
